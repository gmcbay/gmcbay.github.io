Transmat

An accessibility tool for quick text-chat-based callouts while playing Destiny 2 on console platforms like Xbox and Playstation.

Using an Android phone or tablet supporting Bluetooth LE (which most modern phones and tablets do) and a $30 Adafruit Feather 32u4 Bluefruit LE connected to your Xbox or Playstation console, Transmat allows you to set up quick single-press buttons on your Android device that can send abitrary text strings to Destiny 2's text chat as if you typed them on a physical keyboard.

Ideal for when you need to quickly transmit pre-defined information to your fireteam in activities such as raids but voice chat is not an option for everyone in the fireteam. Much easier than trying to juggle a controller and a physical keyboard while actively playing!

UPDATE: I've made some changes to the Adafruit Feather code to better support PC as target platform for this project. Make sure you are using the latest version of the Adafruit Feather 32u4 code found here: https://github.com/gmcbay/Transmat_32u4/releases/download/Release%2F1.0.1/transmat_32u4.zip. To avoid having your own controller or mouse + keyboard input conflict with the input from Transmat you'll have to temporarily remove your hands from the controller or physical keyboard while the callout is transmitted, but this process is fast so it should still be much faster to use this than to physically type in your callouts.

It comes pre-configured with callouts for the Vow of the Disciple raid but you can also create your own custom callout pages for other raids or activities where information needs to be communicated in a timely fashion.

Currently only characters that exist on a US layout keyboard are supported. If anyone is interested in using a system like this with non-English languages please let me know and I can look into improving localization support, it just isn't something I have needed for my own usage thus far.

Transmat is Open Source. You can find the Android app code here and the Adafruit Feather 32u4 Bluefruit LE code here

Setup - Android

Install Transmat from the Google Play Store (if Google Play is available on your Android device). You can find it here.

If using an Android device (such as an Amazon Fire Tablet) that does not have Google Play Store access, you can sideload the Transmat application. Instructions for how to set up sideloading vary from device to device but you should be able to find information for your specific Android device using a web search for 'sideload apk (your-device-here)', for example 'sideload apk fire hd 7'

If sideloading you can find the current version of the Android app apk here

Acquire - Adafruit Feather 32u4 Bluefruit LE

You probably do not have an Adafruit Feather 32u4 Bluefruit LE device laying around already, so you'll likely have to buy one. These are small Arduino-compatible devices that support Bluetooth LE and use the ATmega32U4 processor which is particularly well suited for impersonating a physical USB keyboard. The best place to buy them is direct from Adafruit here

There are other variants of the Adafruit Feather device but this project requires the 32u4 version (to act as a virtual USB keyboard) with Bluetooth LE support so make sure the Adafruit Feather 32u4 Bluefruit LE device is the one you purchase. If you only plan to use this device for Transmat, go ahead and buy the cheapest option labelled "Loose Headers - Soldering Required". Don't panic (if you aren't into electronics): This project does not actually use the IO pins on the Adafruit Feather 32u4 Bluefruit LE device so despite the description no soldering will actually be required.

If you are in a rush and have Amazon Prime you can also choose to buy the device from Amazon here.

You will also need a USB A (male) to Micro B (male) cable. You may have one of these already. If not you can buy one while buying the Adafruit Feather 32u4 Bluefruit LE either at Adafruit or Amazon. The exactly length of the cable is not important as long as it supports USB A male on one side (this side will plug into your console's USB port) and USB Micro B on the other side (this side will plug into the Adafruit device).

Setup - Adafruit Feather 32u4 Bluefruit LE

Once you have the Adafruit Feather 32u4 Bluefruit LE device and the correct USB cable you will need to do a one-time setup using a PC (Windows or Linux) or Mac to write the Transmat firmware to this device. After the firmware is written to the device you won't need the PC or Mac anymore, so if you don't currently have your own PC or Mac this may be something you can do at school, work or on a friend's computer.

  • To get started plug the Adafruit Feather 32u4 Bluefruit LE device into the PC or Mac using the USB A to Micro B cable.
    Plug the USB-A end into your PC or Mac and the Micro B end into the MicroUSB port of the Adafruit Feather 32u4 Bluefruit LE device.
    If things are connected and the device is getting power from the USB cord you should see a red LED light flashing on the device. This is good.

  • Download and install the Arduino IDE from here and start the Arduino IDE

  • Select the File -> Preferences menu


    •  
  • When the Preferences dialog box pops up, find the area for inserting "Additional Boards Manager URLs"


    •  
  • Cut and paste this URL into that text area: https://adafruit.github.io/arduino-board-index/package_adafruit_index.json

  • Click OK to dismiss the Preferences dialog.

  • Select the Tools->Board->Boards Manager... menu


    •  
  • When the Boards Manager dialog pops up, cut and paste this into the search box text area up top as shown: adafruit feather 32u4
    This will filter the list of boards to install support for to just show the one we are interested in (seen in screenshot).


    •  
  • Click the Install button next to the version number in the bottom right corner to begin installing support for the Adafruit Feather 32u4 board.

  • After the Board Manager has installed support for the Adafruit Feather 32u4 board it is recommended you close & restart the Arduino IDE program to make sure everything is properly loaded.

  • Select the Adafruit Feather 32u4 Board from the Tools->Board->Adafruit Boards->Adafruit Feather 32u4 menu as shown


    •  
  • Now select the serial port to use from the Tools->Port menu.
    If everything worked correctly up to this point the IDE should recognize the Adafruit Feather 32u4 on one of the available serial ports and display the board information after the port name (as seen in below screenshot).
    Make sure to choose the serial port that has the Adafruit Feather 32u4 device attached, it may not be COM5 on your system!


    •  
  • Download the Transmat firmware code from here. Unzip the file to a local directory.

  • Select the File->Open menu in the Arduino IDE


    •  
  • Use the open file dialog to navigate to the directory where you unzipped the code above and select the transmat_32u4.ino file inside.


    •  
  • This will open the Transmat firmware project in its own new IDE window.
    You should now be ready to write the firmware.
    In the IDE window that opened with the Transmat firmware project click the upload button (looks like an arrow pointing right) in the top left corner of the Arduino IDE.


    •  
  • You should see some output at the bottom of the IDE window as the IDE compiles and uploads the code to the device.
    If everything works correctly you should eventually see a "Done uploading." message.
    If the upload fails for any reason the first time, try again. If it continues to fail feel free to contact me at george@mcbay.net and I will attempt to help you figure it out.


    •  
  • Once the firmware is done uploading you can unplug the Adafruit device from the PC or Mac you are using and use the USB cable to plug the device into a USB port on your Xbox or Playstation console.
     
    A case is not required for the Adafruit Feather 32u4 Bluefruit LE but if you'd like one, Adafruit has 3D printable files on Thingiverse
    They offer a lot of case variations but the two files you'd generally want to use for this project are feather-case.stl for the main case bottom and feather-top.stl for the top of the case.

    At this point you should be ready to connect the Android app to the device.
     
    Run the Android app on your Android phone or tablet and click the Bluetooth icon in the upper-right corner to attempt to connect.
     
    The first time you attempt to connect you will have to answer some prompts to give the app access to Bluetooth LE.
     
    In order to access Bluetooth LE the app requires location access on older versions of Android, this is normal.
     
    The app doesn't store or transmit any location data.
     
    Once connected you can send your callouts!
